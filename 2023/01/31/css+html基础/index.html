开发前须知

快捷键

  • F12 开发者模式lll
  • Ctrl+r （）刷新。硬性重新加 载：不用缓存进行加载
  • alt+b vscode快速在浏览器中预览网页。
  • ctrl+enter 直接到下一行.
  • Ctrl+alt+↓ 复制这行到下一行
  • 一次选中多个内容：
    • alt+鼠标点击
    • alt+shift+鼠标拖动

浏览器内核

  • 一套代码适配多个内核。
  • trident，gecko,presto,webkit(移动端)，blink（谷歌）

HTML

标签

head标签

meta

  • 设置字符编码
  • 属性:char(字符编码)+set(设置)

style

设置样式

link

引入外部css,网站图标.

body标签

h

  • 可以促进搜索引擎排名的优化(SEO).
  • 一个页面只能有一个h1标签

code

  • 用于显示代码

字符实体

比如&nbsp;代表空格或者使用字符的字符编号。

span

默认情况与普通文本没区别

用于区分普通文本和特殊文本

img

/正斜杠。全用正斜杠

相对路径：”../img/test.jpg”。..代表根目录

  • 图片格式：png透明

一般设置图片高度使用css.如果设置了width自动通过设置图片的宽高比来设置图片的高度。

  • px:像素

a

超链接

target属性

缩短超链接长度的方法

锚点连接：点击某个元素，跳转到某个位置。

伪连接：没有跳转到任何页面，用来执行js代码。

超连接可以嵌套图片

iframe

网页之中嵌套网页

url

CSS

mdn web官方文档非常好。

caniuse 网站可以查阅，某些css属性能不能用。

三种引入方式

内联样式表（inline）

文档样式表

  • 结构与样式分离

外部样式表

  • 样式写在css文件里

  • 最好也设置编码

选择器

通用选择器(不建议使用，影响效率)

类型选择器

  • 一个元素可以有多个类 。类用空格分隔

  • 类规范：1.见名知意2.用类名 “-”连接

id 选择器

  • id名字不要重复

属性选择器

  • ```css
    #选择所有属性
    [title]{
    color:red;
    }#限定某一元素内容进行选择
    [title=”小标题”]{
    color:red;
    }

子选择器

  • ```css
    #选择div 后面直接的span
    div>span{
    color:red;
    }

选择器组

交集选择器

并集选择器

伪类与伪元素

不需要在元素上加上class,通过伪类来选择。

  • 元素状态伪类
  • 动态伪类
  • 目标伪类
  • 语言伪类

Emmet语法

css特性

常见css属性

color

  • 前景色(包括文本颜色，边框颜色等)

文本属性

font-size

  • 文字大小

background-color

  • 背景色

text-decoration

  • none:无装饰线。
  • underline：下划线。
  • overline：上划线。
  • line-through:中划线。

letter-spacing

  • 字母之间的间距

word-spacing

  • 单词之间的间距

text-transform

  • 大小写转化

  • capitalize:首字母大写。

  • uppercase：所有字母大写。

  • lowercase:所有字母小写。

  • none:无影响。

JS

行为，样式，结构三者分离——别写行间样式和js

变量的本质就是一个表

HTML怎么写，js就怎么写 ，但是有如下特例。

  • font-size 改成 fontSize.-换成驼峰记名法

  • class 改成className

事件

事件驱动：当用户进行某种操作的时候，浏览器会主动通知你。

  • 简单、方便、性能高
  • 都是on开头
事件名
onmouseover 光标覆盖
onmouseout 光标离开
onfocus 有焦点

函数

  • ()——执行一个函数

  • 变量=函数名 ——给变量赋值的是函数体

  • 变量=函数名() ——给变量赋值的是函数运行的返回值

  • 元素1.事件1=function(){

    调用函数( )

    }

  • 函数传参

    • 东西会变

undefined 的意义

  • 未定义。或者定义了，但是没赋值。

  • 变量没赋值

  • 函数返回值没写/没return

  • 函数的参数没传值进去

代码执行顺序

  • 正常情况——从上到下
  • 定义——不执行
  • 调用——执行

回调函数

  • 本质上是普通函数。用来区分
  • 把一个函数交给别人，什么时候调用，由他人把握。
函数分类 定义 调用
回调函数 我定义 其他人调用（函数名()）
普通函数 我定义 我调用()

number

  • 转数字

数组

  • length 长度

  • 建议：每个数组只装一类东西

document.getElementsByTagName(‘div’)

  • 获取页面中所有div

  • 得 到的是数组

作用域

全局（能不用尽量不用），

  • 因为能在任何地方都可以使用

  • 任何人都可能修改。

  • 容易重名

  • 性能低

局部

作用域是以函数为单位的，只能在函数内使用。

var （不建议使用）

  • 缺点：变量重复声明

  • 不能限制修改

  • 不支持块级作用域

let

  • 不允许重复声明

const

常量

  • 不允许重复声明

  • 限制你修改
